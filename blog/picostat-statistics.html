Introducing Picostat by North Penn Networks

Picostat is a statistical web application used to manage datasets. It can also be used to perform basic statistical operations like:

  • Computing the mean
  • Computing the standard deviation
  • Plotting various histograms, dotplots, and stem-and-leaf plots
  • Correlation and regression analysis

Picostat can also be used as a homework helper which shows students how to:

  • How to compute the mean
  • How to create a bar plot and stacked barplot
  • How to create a dotplot
  • How to create a pie chart
  • How to compute the median

Picostat is a project that has been in development for sometime. It had originally been created by Tetragy Limited. Development was then taken over by North Penn Networks.

You can find out more about the Picostat Statistical Web Application on it home on the web. Feel free to follow North Penn Networks on Twitter.
